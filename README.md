## Semantic recommendations at the speed of light.
Sentio is a high-performance, content-aware recommendation engine built for the modern web. By marrying the deep semantic understanding of RoBERTa with the lightning-fast vector search of LanceDB, Sentio delivers real-time, personalized suggestions directly through a robust Django backend.

**Deep Semantic Matching:** Leverages sentence-transformers and RoBERTa to understand the meaning behind content, not just keywords.
**Serverless Vector Search:** Powered by LanceDB for sub-millisecond similarity lookups without the overhead of a complex database cluster.
**Production-Ready API:** A fully integrated Django application providing secure, scalable, and easy-to-consume recommendation endpoints.
**Hybrid Storage:** Seamlessly combines structured metadata in SQL with high-dimensional embeddings in a vector store.
